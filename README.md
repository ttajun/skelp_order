# bbdr-cshop

bbdr-cshop is python exectuable command for shopping-mall crawling.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install bbdr-cshop

```bash
pip install bbdr-cshop
```

## Usage

```console
# execute
[user@dev]$ bbdr-cshop -h
```
