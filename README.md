# skelp_order

skelp_order is python exectuable command for selenium crawling.

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install skelp_order

```bash
pip install skelp_order
```

## Usage

```console
# execute
[user@dev]$ skelp_order -h
```
